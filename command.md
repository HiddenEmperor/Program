
**git 初始化**
```bash
git init
```

**git 状态查看**
```bash
git status
```

**git 添加文件（. 表示添加所有文件）**
```bash
git add <filename>
git add .
```

**git 日志查看**
```bash
git log
git log --oneline
```

**git 提交本地**
```bash
git commit -m "first commit"
```

**git 版本回退**
```bash
git checkout <commit_id>
```

**git 提交远端仓库**
```bash
git push origin master
git push origin dev
```

**git 拉取远端仓库**
```bash
git pull origin master
```

**git 分支查看**
```bash
git branch
```

**git 切换主分支**
```bash
git checkout master
```
